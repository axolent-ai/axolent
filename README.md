# Jarvis-LITE

Open-Source AI-Assistent via Telegram + eigenstaendige Apps. Lokal ausgefuehrt, multi-provider (Claude, GPT, Gemini, lokale Modelle), mit persistenter Memory.

## Repository-Struktur

- bridge/     FastAPI Bridge-Service (Backend, Provider-Routing, Memory, Tools)
- mini-app/   Telegram Mini App (Frontend in Telegram WebApp)
- desktop/    Desktop-App via Tauri (Rust + Webview)
- shared/     Geteilte UI-Komponenten zwischen Mini App und Desktop
- docs/       Technische Doku, API-Referenz, Setup-Anleitungen
